# Shop Y2K - Homepage E-commerce Générée avec Claude AI

Projet expérimental réalisé afin de tester les capacités d'une IA générative dans la conception d'une interface e-commerce moderne.

L'objectif n'était pas seulement de produire une page web, mais également de comprendre comment collaborer efficacement avec une intelligence artificielle dans un contexte de développement web.

## Objectifs

- Expérimenter le développement assisté par IA
- Créer une landing page e-commerce moderne
- Travailler le design inspiré de l'esthétique Y2K
- Tester les bonnes pratiques de prompting

## Technologies utilisées

- HTML
- CSS
- JavaScript
- Claude AI

## Fonctionnalités

- Hero section immersive
- Mise en avant de produits
- Design responsive
- Interface inspirée des années 2000

## Ce que j'ai appris

- Structurer un prompt technique
- Collaborer avec une IA dans un projet web
- Analyser et améliorer du code généré
- Adapter une interface aux standards actuels

## Remarque

Ce projet est avant tout un laboratoire d'expérimentation autour du développement assisté par intelligence artificielle.
